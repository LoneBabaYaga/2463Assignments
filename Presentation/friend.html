Mole Mayhem (Whack-a-mole) Project Documentation

Project Proposal by Marshall McCurdy

Description

"Mole Mayhem" is a whack-a-mole style game where players dive into the world of a vibrant garden and are tasked with protecting it from mischievous moles. Players must whack the moles as they pop up from the ground to earn points, but must avoid hitting any friendly creatures to avoid penalties. The game ends if the player misses three moles in a row and gets progressively faster.

Graphics

Sprites for hammer, moles, and friendly creatures are used to enhance the visual appeal and interaction of the game.

Sound

The game features various sounds including the classic whack-a-mole sound, "you win" and "you lose" music, background music, and specific sounds when hitting a friendly creature.

Hardware Input

Players interact with the game using a mouse to click and whack the moles as they appear.

Hardware Output

The game uses 2 LED lights to indicate game status: green when you win, and red when you lose.
