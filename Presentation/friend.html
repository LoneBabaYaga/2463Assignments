Mole Mayhem (Whack-a-mole) Project Documentation

Project Proposal by Marshall McCurdy

Description

"Mole Mayhem" is a whack-a-mole style game where players dive into the world of a vibrant garden and are tasked with protecting it from mischievous moles. Players must whack the moles as they pop up from the ground to earn points, but must avoid hitting any friendly creatures to avoid penalties. The game ends if the player hits 3 friendly creatures. You win by hitting 10 moles.

Graphics

Images for moles, and friendly creatures are used to enhance the visual appeal and interaction of the game.

Sound

The game features various sounds including the classic whack-a-mole sound, "you win" and "you lose" music, background music, a dog bark sound when hitting the friendly, a sound for when you miss a hit, and also a sound when you hit a mole.

Hardware Input

The game has no hardware input except the use of the mouse for the game. There was plans for a joystick, but did not get to it.

Hardware Output

There is no hardward output, however, it can be implemented for the future. The idea was to have the button turn green when a mole is hit, and turn red when a friendly creature is hit.
